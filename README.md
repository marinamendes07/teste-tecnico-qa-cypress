# Teste Técnico QA – Cypress | BDD

Este repositório contém a implementação do teste técnico para a vaga de **QA**, utilizando **Cypress**, **Gherkin** e **Cucumber (BDD)**.

## Tecnologias utilizadas
- Cypress
- Gherkin
- Cucumber (cypress-cucumber-preprocessor)
- JavaScript

## Estrutura do projeto
- `features/` → Cenários escritos em Gherkin  
- `steps/` → Step Definitions (Cucumber)  
- `pages/` → Page Objects  
- `elements/` → Mapeamento de elementos  
- `e2e/` → Testes automatizados com Cypress

## Organização dos testes
- Questões 1 a 4: Testes utilizando Gherkin com Cypress
- Questão 5: Teste utilizando Cucumber + Cypress (BDD)

## Observações
Os testes foram desenvolvidos de forma conceitual, seguindo as instruções do teste técnico.  
Onde não havia identificadores definidos, foram utilizados seletores textuais ou estruturais.
